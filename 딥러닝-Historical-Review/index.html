딥러닝 Historical Review

Introduction

  • 인공지능의 정의: 사람의 지능을 모방하는 것.
  • Key Components of Deep Learning
    1. Data (The data that the model can learn from)
    2. Model (The model how to transform the data)
    3. Loss (The loss function that quantifies the badness of the model)
    4. ALgorithm (The algorithm to adjust the parameters to minimize the loss)

Data

  • 데이터는 해결하고자 하는 문제의 타입에 의존하게 된다.
    • Classification(분류를 위한 데이터셋), Semantic Segmentation(이미지 픽셀별 분류), Detection(이미지에 물체가 있을 때 물체의 바운딩박스를 찾고자 하는 것), Pose Estimation(사람의 3차원 혹은 2차원 스켈레톤 정보), Visual QnA(사진과 질문이 주어졌을 때 답을 하는 것)

Model

  • 같은 데이터 혹은 태스크가 주어졌다고 하더라도 모델의 성질에 따라서 퍼포먼스의 차이가 있을 수 있음.
  • 결과를 잘 나타내기 위한 테크닉들이 존재

Loss

  • 모델과 데이터가 정해져 있을 때 모델을 어떻게 학습할 지에 대한 것임.
  • weight를 업데이트하는 방향성을 제시함.
  • 회귀에서는 대표적으로 MSE, 분류에서는 대표적으로 CrossEntropyLoss, 확률형태스크에서는 MLE 등.
  • 이루고자 하는 것의 Proxy(근사치)에 불과하다. 일반적으로 태스크를 봤을 때 loss의 값이 줄어든다는 것을 목적으로만 할 수는 있지만, loss를 줄이는 것 만으로는 우리가 원하는 결과를 얻지 못할 수도 있음(데이터에 noise가 많은 경우 등)

Optimization Algorithm

  • 데이터와 모델, Loss가 정해져 있을 때 Network를 어떻게 줄일지에 대한 것임.
  • Momentum, Nag, Adagrad 등을 사용하는 방법론들이 존재한다.

Historical Review

Deep Learning’s Most Important Ideas - A Brief Historical Review (Denny Britz, 2020-07-29)

  1. 2012 - AlexNet

  2. 2013 - DQN

    DQN은 알파고의 학습방법론인 강화학습이다. 오늘날의 딥마인드가 있게 한 논문.

  3. 2014

    Encoder/Decoder

    • NLP의 번역 Task를 해결하기 위한 방법이다.

    • Sequence to Sequence 모델임.

    Adam

    • 대부분 아담을 그냥 쓰는데, 이유는 그냥 결과가 잘 나와서이다.

  4. 2015 - GAN, ResNet

    GAN

    ResNet

    • 왜 딥러닝이 딥러닝으로 불렸는지를 증명한 그런 모델이다.
    • 네트워크를 깊게 쌓을 수 있는 패러다임을 제시함.

  5. 2017 - Transformer

    • Attention is All You Need

  6. 2018 - Bert

    • Fine-Tuned NLP Model들이 발전하기 시작.

  7. 2019 - Big Language Models(GPT-X)

    • OpenAI - GPT-3, an autoregressive language model with 175 billion parameters

  8. 2020 - Self-Supervised Learning

    • SimCLR: a simple framework for contrastive learning of visual representations
    • 라벨을 모르는 Unsupervised 데이터를 활용할 수 있는 방법론.
